# useless-skills

可独立安装和更新的 Codex Skills。

## Skills

- [debug-third-party-api](skills/debug-third-party-api/README.md)：实测第三方 API 文档并生成单文件 HTML 调试报告。

## 安装

把目标 Skill 的 GitHub 目录地址交给 Codex：

```text
使用 $skill-installer 安装这个 Skill：
https://github.com/AugustusHu/useless-skills/tree/main/skills/debug-third-party-api

如果默认下载遇到本机 SSL 证书错误，请改用安装器的 Git 模式，
不要关闭 TLS 校验。
```

具体使用和更新方式见对应 Skill 的 README。
